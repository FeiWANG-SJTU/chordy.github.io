git 基本操作

#git 基本操作

git add –all track 所有
git vim file 进入文本编辑
esc 退出编辑
:WQ！ 保存改动并退出
git status -s 查看状态
git add file 添加更改
git diff –cached 已缓存的改动
git diff 未缓存的改动
git diff HEAD 所有改动
git commit -m ‘my hola mudo changes’ 记录缓存内容的快照
-m 在命令行中提供提交注释 如果没有-m 默认打开vim 后面是
自己的改动内容
git log 查看提交过的更改记录
git pull 从远端服务器提取新数据
git push https://github.com/chordy/DNAcomp.git
